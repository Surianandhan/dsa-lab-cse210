# Data Structures Lab in C (CSE210)

This repository contains Data Structures lab programs implemented in C as part of the CSE210 course.

## Overview

The programs in this repository demonstrate fundamental data structures and algorithms using the C programming language. These implementations focus on building a strong understanding of core concepts and problem-solving techniques.

## Topics Covered

- Arrays
- Linked Lists (Singly / Doubly)
- Stacks
- Queues
- Searching Algorithms (Linear, Binary)
- Sorting Algorithms (Bubble, Selection, etc.)
- Recursion

## Project Structure

Each program is stored as an individual `.c` file with meaningful names.

Recommended structure (can be organized later):

dsa-lab-cse210/
│
├── arrays/
├── linked-list/
├── stack/
├── queue/
├── sorting/
├── searching/

## How to Run

1. Compile the program:
   gcc filename.c

2. Run the program:
   ./a.out

## Purpose

- Understand core data structures
- Practice algorithm implementation
- Improve logical and problem-solving skills
